A professional coaching website project repository that contains two separate Next.js website builds for educational institutes, organized under one parent workspace for easy comparison, improvement, and future merge into a single final product.

This project is ideal for building modern coaching institute websites with sections such as admissions, teachers, gallery, testimonials, contact information, and student registration.

## 💡 Why This Project Matters

- Modern coaching website development
- Next.js and React-based frontend architecture
- Clean educational institute UI design
- Easy comparison between two website versions
- Strong foundation for a final production-ready website

## 🗂️ Project Structure

```text
CoachingWebsite/
├── website1/
│   └── First coaching website version
├── website2/
│   └── Second coaching website version
├── README.md
└── .gitignore
```

## What Is Inside?

- website1: A polished coaching website with sections for admissions, gallery, teachers, and more.
- website2: A second design version with its own layout, components, and content structure.

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS
- JavaScript / JSX
- CSS Modules
- Node.js

## 🚀 Setup Instructions

### Run Website 1

```bash
cd website1
npm install
npm run dev
```

### Run Website 2

```bash
cd website2
npm install
npm run dev
```
