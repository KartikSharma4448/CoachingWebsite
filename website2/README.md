# Atharva Classes - Professional Website

A high-end, professional school website built with **Next.js**, featuring a playful and modern UI inspired by the Kidza template. This project is designed to provide a premium user experience for students, parents, and faculty.

## 🚀 Features

- **Modern UI/UX**: Playful color palette, wavy dividers, and smooth animations.
- **Fully Responsive**: Optimized for all devices (Mobile, Tablet, Desktop).
- **Next.js 14**: Fast performance and SEO-friendly structure.
- **Interactive Sections**:
  - Hero with floating elements
  - Animated scroll counters (Stats)
  - Program cards with hover effects
  - Professional Contact & Newsletter forms
- **Dynamic Assets**: Integrated high-quality school-themed images.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Library**: [React](https://reactjs.org/)
- **Styling**: Vanilla CSS Modules (for maximum flexibility)
- **Animations**: CSS Keyframes & Framer Motion (Ready for expansion)
- **Icons**: Emoji-based and icon-ready structure.

## 🏃 Getting Started

### 1. Install Dependencies
First, make sure you have [Node.js](https://nodejs.org/) installed. Then, run:

```bash
npm install
```

### 2. Run the Development Server
To start the project locally, run:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 3. Build for Production
To create an optimized production build:

```bash
npm run build
```

## 🌐 Deployment

The easiest way to deploy this app is to use the [Vercel Platform](https://vercel.com/new).

1. Push your code to a GitHub/GitLab/Bitbucket repository.
2. Import the project into Vercel.
3. Vercel will automatically detect Next.js and deploy your professional school website.

## 📁 Project Structure

- `app/`: Next.js App Router, global styles, and layout.
- `components/`: Modular React components (Hero, About, Programs, etc.).
- `public/images/`: High-resolution school assets and images.
- `package.json`: Project dependencies and scripts.
